# ♡ SQL Fundamentals

## ✦ Overview

This repository contains SQL projects developed during the **Data and Artificial Intelligence Engineering** program at **Faculdade Infnet**.

The projects focus on fundamental SQL concepts, including data retrieval, filtering, logical operators and query construction using SQLite.

---

## ✦ Technologies

* SQL
* SQLite
* SQLite Online

## ✦ Repository Structure

```text
projects/
├── project-01-basic-queries
├── project-02-sql-queries
└── ...
```

Each project contains:

* SQL scripts
* Project documentation
* Exercises organized by topic

## ✦ Topics Covered

* Logical operators (AND, OR, NOT)

* Query organization
