# ♡ Project 01 — Basic SQL Queries

## ✦ Overview

This project was developed individually as part of the **Data and Artificial Intelligence Engineering** program at **Faculdade Infnet**.

The objective was to practice fundamental SQL querying techniques using SQLite through a series of progressively more complex exercises.

---

## ✦ Objective

Develop SQL queries to retrieve, filter and organize data stored in a relational database, applying the fundamental concepts of SQL syntax and logical operators.

---

## ✦ Technologies

* SQL
* SQLite
* SQLite Online

---

## ✦ Topics Covered

The exercises explore fundamental SQL concepts, including:

* SELECT statements
* WHERE clause
* Comparison operators
* Logical operators (AND, OR, NOT)
* ORDER BY
* Aggregate functions
* Filtering records
* Data organization
* Query optimization fundamentals
